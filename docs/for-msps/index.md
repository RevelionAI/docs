# Revelion for MSPs

Revelion lets Managed Security Service Providers deliver autonomous AI penetration testing to multiple clients from a single platform — no separate accounts, no manual coordination, no per-client tooling.

## Why MSPs use Revelion

Traditional pentesting delivery is slow and expensive. Scheduling, scoping, tooling, reporting, and remediation tracking all require manual effort. Revelion replaces that pipeline with an autonomous agent that runs end-to-end tests on demand, generates professional PDF reports, and exposes results to clients through a branded self-service portal.

### Key benefits

**Scale across clients**
Run concurrent scans across multiple client environments. Each scan is isolated, scoped, and attributed to the correct client automatically.

**White-label everything**
Reports carry your branding — company name, logo, colours, header and footer text. Clients never see Revelion. Per-client branding overrides let you differentiate presentation per engagement.

**Client self-service portal**
Share a PIN-protected portal link with each client. They view their findings, reports, and scan history without needing a Revelion account.

**Recurring scheduled scans**
Set up automated scan schedules per client. Deliver continuous security testing as a managed service without manual intervention.

## Getting started

1. Upgrade to the MSP plan in **Settings → Membership**
2. Add clients in the **MSP** section
3. Configure [white-label branding](white-label.md) in **Settings → Report Branding**
4. Run scans attributed to each client
5. Share [client portal links](client-portal.md) to deliver results
